# 🐱 Lucky - 交易复盘工具

一款专业但不冷冰冰的交易复盘工具，采用深色主题和小猫主题图标，帮助交易者记录、分析和改进交易策略。

![Lucky Trading Journal](assets/icons/favicon.png)

## ✨ 功能特性

### 📊 9 大核心模块

1. **每日交易复盘记录**
   - 支持语音输入（Web Speech API）和打字输入
   - 记录：日期、品种、方向、价格、仓位、盈亏、复盘内容
   - 按日期倒序排列，支持日期筛选

2. **定期报告（周报 + 月报）**
   - 自动生成交易统计报告
   - 包含：交易次数、胜率、盈亏比、总盈亏、最大单笔盈亏
   - 支持导出为图片分享

3. **心情记录**
   - Emoji 情绪选择：FOMO / 恐惧 / 冷静 / 贪婪 / 焦虑 / 平静 / 报复性交易
   - 情绪与交易结果关联分析
   - 支持自定义情绪标签

4. **交易标签系统**
   - 预设标签：计划内、追高、冲动单、止损执行、扛单、加仓、减仓等
   - 支持自定义标签
   - 统计分析每种标签的胜率

5. **交易前检查清单**
   - 可自定义检查项
   - 新建交易时弹出确认
   - 统计完成检查 vs 未完成检查的胜率差异

6. **亏损模式识别**
   - 自动分析亏损规律
   - 发现：连续亏损后行为、时间模式、标签关联等
   - 数据可视化展示

7. **截图/图表绑定**
   - 每笔交易支持上传 K 线截图
   - 复盘列表可查看大图
   - 截图与交易记录绑定

8. **纪律打卡**
   - 每日打卡：是否按计划执行
   - 连续执行天数统计
   - 日历视图展示

9. **资金曲线**
   - 自动计算并绘制净值曲线
   - 支持多时间范围选择
   - 标注最大回撤区间

### 🔐 用户系统

- **注册/登录**：邮箱 + 密码
- **云端同步**：Supabase BaaS
- **离线支持**：未登录时数据暂存 localStorage
- **数据导入/导出**：JSON 格式备份

## 🚀 快速开始

### 1. 配置 Supabase

1. 访问 [Supabase](https://supabase.com) 创建项目
2. 按照 `SUPABASE_SETUP.md` 配置数据库
3. 在 `js/app.js` 中填入你的 Supabase URL 和 Anon Key

### 2. 本地运行

```bash
# 使用 Python 的简单 HTTP 服务器
cd lucky-trading-journal
python -m http.server 8080

# 或使用 Node.js 的 http-server
npx http-server -p 8080
```

打开浏览器访问 `http://localhost:8080`

### 3. 添加图标

将图标文件放置到 `assets/icons/` 目录：

```
assets/icons/
├── favicon.png          # 浏览器标签图标
├── nav-dashboard.png    # 仪表盘图标
├── nav-record.png       # 记录图标
├── nav-list.png         # 列表图标
├── nav-analysis.png     # 分析图标
├── nav-report.png       # 报告图标
├── nav-curve.png        # 曲线图标
└── nav-settings.png     # 设置图标
```

## 📝 使用指南

### 首次使用

1. 打开应用，可选择：
   - 注册/登录账户（数据同步到云端）
   - 跳过登录先体验（数据保存在本地）

2. 在设置页面：
   - 设置初始资金
   - 自定义交易标签
   - 自定义检查清单

3. 开始记录交易：
   - 点击"记录"页面
   - 填写交易信息
   - 勾选检查清单
   - 选择情绪和标签
   - 保存记录

### 日常流程

1. **开盘前**：设置今日交易计划
2. **交易中**：按计划执行
3. **收盘后**：
   - 纪律打卡
   - 记录当日交易
   - 上传 K 线截图
   - 复盘笔记

### 定期复盘

- **每周日**：查看周报，总结本周交易
- **每月底**：查看月报，分析月度表现
- **持续监控**：关注资金曲线和亏损模式

## 🔒 数据安全

- 所有数据存储在 Supabase PostgreSQL 数据库
- Row Level Security (RLS) 确保数据隔离
- 支持本地 JSON 导出备份
- 图片存储在 Supabase Storage

## 🐛 常见问题

**Q: 可以在没有网络的情况下使用吗？**
A: 可以！数据会保存在浏览器本地，联网后自动同步。

**Q: 如何更换设备？**
A: 登录相同账户即可自动同步所有数据。

**Q: 数据会丢失吗？**
A: 建议定期导出 JSON 备份，以防万一。

**Q: 支持哪些浏览器？**
A: Chrome, Firefox, Safari, Edge 最新版本均支持。
